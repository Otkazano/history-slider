### Лендинг «Исторические даты»‎

Проект представляет собой страницу с анимацией.

### Установка и запуск проекта:

Клонировать репозиторий:

```bash
git clone https://github.com/Otkazano/history-slider.git
```

Установаить зависимости:

```bash
npm install
```

Запустить проект:

```bash
npm run start
```
